# React Counter Application

This project is a simple interactive web application built using React for my Computer Science 215 final project. The purpose of this project is to demonstrate how React can be used to create dynamic and interactive user interfaces.

## Features
- Increase the counter
- Decrease the counter
- Reset the counter to zero
- Updates the page without reloading

## Concepts Used
- React components
- useState hook
- Event handling with onClick
- Dynamic rendering

## Technologies Used
- React
- JavaScript
- HTML
- CSS
